### Workflow Library of Anar Framework

#### Usage Guide: 

add the following dependency to pom.xml 


```
		<dependency>
			<groupId>af.gov.anar.lib</groupId>
			<artifactId>anar-lib-workflow</artifactId>
			<version>${project.version}</version>
		</dependency>
```
